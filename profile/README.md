# ARCS - Automated Record Custody Standard

<p align="left">
  <img src="https://raw.githubusercontent.com/arcs-standard/.github/main/profile/arcs-banner.png" alt="ARCS" width="50%" />
</p>

Published by [Vega Commons Project, Inc.](https://vegacommons.org)

ARCS defines governance controls for the lifecycle, custody, retention, preservation, and deletion of interaction records generated by automated systems during human-directed operation.

**Standard text and reference materials:** [arcsstandard.org](https://arcsstandard.org)
